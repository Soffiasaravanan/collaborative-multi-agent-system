# Collaborative Multi-Agent System

## Overview
This project implements a Collaborative Multi-Agent System using Python, Ollama, and the Llama 3.2 Large Language Model.

## Features
- Manager Agent
- Researcher Agent
- Designer Agent
- Frontend Agent
- Backend Agent
- Tester Agent
- Writer Agent

## Technologies Used
- Python
- Ollama
- Llama 3.2
- Visual Studio Code

## Project Structure

```
├── agents.py
├── main.py
├── README.md
```

## Installation

1. Install Ollama.
2. Pull the model:
```
ollama pull llama3.2
```

3. Install the Python package:
```
pip install ollama
```

4. Run:
```
python main.py
```
